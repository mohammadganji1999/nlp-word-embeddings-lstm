# NLP Project: Word Embeddings & Sentiment Analysis

This project demonstrates the use of pre-trained GloVe word embeddings and deep learning for NLP tasks.

## Features
- Word analogy solving using GloVe vectors
- t-SNE visualization of word embeddings
- Document similarity with cosine similarity
- LSTM-based sentiment analysis classifier in PyTorch

## Technologies
Python, PyTorch, Gensim, scikit-learn, NumPy, Matplotlib

## How to run
pip install -r requirements.txt  
python main.ipynb
